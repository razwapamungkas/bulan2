## Latihan Dasar JavaScript

### Tujuan

Pada latihan ini, peserta diharapkan mampu memahami dan menerapkan konsep dasar JavaScript berikut:

* Sintaks JavaScript
* Statements
* Expressions
* Variable
* Data Type
* JavaScript Operator

---

## Instruksi

Buat sebuah file bernama `script.js` dan kerjakan seluruh tugas berikut.

---

## 1. Variable & Data Type

Buat beberapa variabel yang menyimpan data berikut:

* Nama lengkap
* Umur
* Tinggi badan
* Status sebagai pelajar/programmer (boolean)
* Hobi favorit

Tampilkan seluruh nilai variabel menggunakan `console.log()`.

## 2. Memahami Sintaks JavaScript

Tuliskan minimal **5 baris kode JavaScript** yang valid.

Contoh:

```javascript
let nama = "Budi";
let umur = 20;
console.log(nama);
console.log(umur);
let kota = "Jakarta";
```

Tambahkan komentar pada setiap baris untuk menjelaskan fungsi kode tersebut.

---

## 3. Statements

Buat minimal **5 statement JavaScript** yang berbeda.

Contoh statement yang dapat digunakan:

* Deklarasi variabel
* Inisialisasi nilai variabel
* Assignment nilai baru ke variabel
* Pemanggilan `console.log()`
* Operasi perhitungan yang disimpan ke variabel

Berikan komentar untuk menjelaskan masing-masing statement.

---

## 4. Expressions

Buat dan tampilkan hasil dari expression berikut:

### Arithmetic Expression

```javascript
10 + 5
```

### String Expression

```javascript
"Hello" + " World"
```

### Comparison Expression

```javascript
20 > 10
```

### Logical Expression

```javascript
true && true
```

Tampilkan hasil seluruh expression menggunakan `console.log()`.

---

## 5. JavaScript Operator

Demonstrasikan penggunaan operator berikut:

### Arithmetic Operator

* `+`
* `-`
* `*`
* `/`
* `%`

### Assignment Operator

* `=`
* `+=`
* `-=`

### Comparison Operator

* `==`
* `===`
* `!=`
* `>`
* `<`

### Logical Operator

* `&&`
* `||`
* `!`

Tampilkan hasil dari setiap operasi menggunakan `console.log()`.

---

## 6. Program Kalkulator Sederhana

Buat dua variabel yang berisi angka.

Lakukan operasi berikut:

* Penjumlahan
* Pengurangan
* Perkalian
* Pembagian
* Modulus (sisa hasil bagi)

Simpan setiap hasil operasi ke dalam variabel yang berbeda, kemudian tampilkan hasilnya.

## Bonus

### Biodata Digital

Buat variabel yang menyimpan:

* Nama
* Umur
* Kota Asal
* Hobi
* Bahasa Pemrograman Favorit

Kemudian tampilkan dalam format berikut:

```text
=== BIODATA DIGITAL ===

Nama: Ahmad
Umur: 21
Kota Asal: Surabaya
Hobi: Bermain Futsal
Bahasa Pemrograman Favorit: JavaScript
```

Tambahkan minimal **3 expression** yang menggabungkan beberapa data dari biodata tersebut.

Contoh:

```javascript
let perkenalan = "Halo, nama saya " + nama;
```
